# Active Exec Plans

Store in-progress plan files in this directory. Name them `YYYY-MM-DD-short-topic.md` and keep them current while work is happening.

Every active plan should:

- use the template in [../../PLANS.md](../../PLANS.md);
- link back to a product spec, design doc, or debt item;
- describe verification steps before implementation begins.
