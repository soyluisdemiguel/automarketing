# Completed Exec Plans

Move finished or superseded plan files into this directory to preserve implementation history without cluttering active work.

Before moving a plan here:

- update linked specs and design docs;
- add any leftover debt to [../tech-debt-tracker.md](../tech-debt-tracker.md);
- mark the plan status as completed, cancelled, or superseded.
